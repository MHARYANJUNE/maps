🗺️ Map Functionality

To test the map:

  1. Open index.html in your browser
  2. Wait for the loading screen to disappear
  3. Try the following features:
    • 📍 Click the locate button (first button on the left)
    • 📌 Click the marker button to enable marker placement mode
    • 🚗 Click the route button to enable routing mode
    • ✏️ Click the manage button to edit/remove markers
    • 🗑️ Click the clear button to remove routes
    • ❓ Click the help button in the header
    • ⚙️ Click the settings button in the header
